# 계획이 다 있구나 - 계.다.있
학생들과 선생님들을 위한 스케줄 관리 앱
- 기능
  - 시간표 생성 및 과목 추가
  - 과외 선생님을 위한 정산 기능(금액 및 입금 여부 확인)
  - 드래그 앤 드롭을 통해 과목 시간을 쉽게 수정
